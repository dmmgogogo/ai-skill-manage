# AI Skill Manager — M2 实施计划（编辑闭环）

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 把 M1 的"只读浏览器"扩展为"能改、能保存、能感知外部变化"的完整编辑器。完成后 user 能在 App 内编辑任意 skill 的 raw markdown，⌘S 写盘，App 重新激活时自动同步外部改动（不打断正在编辑的内容）。

**Architecture:** 在 `SkillRepository` 协议加 `save(item:content:)` 方法；新增 `DetailEditorVM` 持有编辑缓冲；`DetailView` 把只读 `Text` 换成可编辑 `TextEditor`；`AppRoot` 监听 `NSApplication.didBecomeActiveNotification` 触发刷新。文件原子替换通过 `FileManager.replaceItemAt(...)`。

**Tech Stack:** Swift 5.9 / SwiftUI / @Observable / FileManager / NotificationCenter / XCTest（已铺好）

**Scope cuts (deliberate)**:
- ❌ **不做语法高亮**：用 SwiftUI 内置 `TextEditor` + monospaced font，简单直白。语法高亮留到 M4 打磨阶段
- ❌ **不做 banner / modal 冲突解决**：spec §7.5 已确认"在编辑时保留内存内容、保存覆盖磁盘"
- ❌ **不做撤销/重做**：`TextEditor` 自带的足够
- ❌ **不做 Find / Replace**：M4 全局搜索时一并

**Spec:** `docs/superpowers/specs/2026-04-28-ai-skill-manager-design.md`，§7.2（保存）/ §7.5（聚焦刷新）/ §8（错误处理）

- [ ] **Step 3: 提取共享 atomic-write helper**

在 `AISkillManager/Infrastructure/` 新建 `AtomicWriter.swift`:

```swift
import Foundation

enum AtomicWriter {
    /// Atomically writes content to `target` via a sibling tmp file + replaceItemAt.
    /// Throws on any write/replace failure. Cleans up tmp file on failure.
    static func write(_ content: String, to target: URL) throws {
        let parent = target.deletingLastPathComponent()
        let tmpName = ".\(target.lastPathComponent).tmp.\(UUID().uuidString)"
        let tmpURL = parent.appendingPathComponent(tmpName)

        let data = Data(content.utf8)

        do {
            try data.write(to: tmpURL, options: [.atomic])
        } catch {
            try? FileManager.default.removeItem(at: tmpURL)
            throw error
        }

        do {
            // replaceItemAt requires the destination to exist; if it doesn't (e.g.,
            // file was deleted externally), we throw rather than create-via-replace
            // to surface the unexpected state to the caller.
            guard FileManager.default.fileExists(atPath: target.path) else {
                try? FileManager.default.removeItem(at: tmpURL)
                throw CocoaError(.fileNoSuchFile)
            }
            _ = try FileManager.default.replaceItemAt(target, withItemAt: tmpURL)
        } catch {
            try? FileManager.default.removeItem(at: tmpURL)
            throw error
        }
    }
}
```

注意：`replaceItemAt` 成功时会自动消耗 tmpURL（移动而非复制）。失败时我们手动清理 tmpURL。

```swift
    /// Replace an existing item in itemsBySource keyed by its source.
    /// Used after save/edit to reflect updated content/mtime without a full rescan.
    func updateItem(_ updated: SkillItem) {
        let key = Self.sourceKey(kind: updated.kind, scope: updated.scope)
        guard var items = itemsBySource[key] else { return }
        guard let idx = items.firstIndex(where: { $0.id == updated.id }) else { return }
        items[idx] = updated
        itemsBySource[key] = items
    }

    /// Look up the repository that owns the given item (by kind + scope).
    /// Returns nil if no repo matches (shouldn't happen for items emitted by repos).
    func repository(for item: SkillItem) -> SkillRepository? {
        let key = Self.sourceKey(kind: item.kind, scope: item.scope)
        return repos.first { Self.sourceKey(kind: $0.kind, scope: $0.scope) == key }
    }
```

```swift
import Foundation
import Observation

enum DetailEditorError: Error {
    case noBoundItem
}

@Observable
final class DetailEditorVM {
    private let store: AppStore

    /// The item currently bound to the editor. Nil when no item is selected.
    private(set) var boundItem: SkillItem?

    /// Editor buffer — what the user is currently editing. Bidirectionally bound to TextEditor.
    var editingContent: String = ""

    /// True when editingContent differs from the bound item's rawContent.
    var isDirty: Bool {
        guard let item = boundItem else { return false }
        return editingContent != item.rawContent
    }

    init(store: AppStore) {
        self.store = store
    }

    /// Load an item into the editor. Discards any unsaved changes.
    func bind(to item: SkillItem) {
        boundItem = item
        editingContent = item.rawContent
    }

    /// Save current buffer to disk via the owning repository, then update AppStore.
    func save() throws {
        guard let item = boundItem else { throw DetailEditorError.noBoundItem }
        guard let repo = store.repository(for: item) else { throw DetailEditorError.noBoundItem }

        let updated = try repo.save(item: item, content: editingContent)
        boundItem = updated
        editingContent = updated.rawContent
        store.updateItem(updated)
    }
}
```

- [ ] **Step 2: 改 AppRoot 注入 editor + 聚焦刷新**

把 `AISkillManager/App/AppRoot.swift` 改成：

```swift
import SwiftUI

struct AppRoot: View {
    @State private var store = AppStore.makeDefault()
    @State private var editor: DetailEditorVM

    init() {
        let s = AppStore.makeDefault()
        _store = State(initialValue: s)
        _editor = State(initialValue: DetailEditorVM(store: s))
    }

    var body: some View {
        NavigationSplitView {
            SidebarView(store: store)
        } content: {
            ItemListView(store: store)
        } detail: {
            DetailView(store: store, editor: editor)
        }
        .frame(minWidth: 1000, minHeight: 600)
        .task {
            await store.loadAll()
            // After initial load, bind editor to currentItem if any.
            if let item = store.currentItem, editor.boundItem == nil {
                editor.bind(to: item)
            }
        }
        .onReceive(NotificationCenter.default.publisher(for: NSApplication.didBecomeActiveNotification)) { _ in
            Task { await refreshOnFocus() }
        }
        .toolbar {
            ToolbarItem(placement: .primaryAction) {
                Button {
                    Task { await store.loadAll() }
                } label: {
                    Label("刷新", systemImage: "arrow.clockwise")
                }
            }
        }
    }

    /// Focus-refresh policy (spec §7.5 simplified version):
    /// - If user is NOT editing (editor.isDirty == false): rescan and silently update.
    /// - If user IS editing: rescan but do NOT touch the bound item; the user's
    ///   in-memory buffer wins, and on save will overwrite whatever's on disk.
    private func refreshOnFocus() async {
        let wasEditing = editor.isDirty
        let editingItemID = editor.boundItem?.id

        await store.loadAll()

        // Re-bind editor only if the user wasn't editing OR the bound item is gone.
        if !wasEditing {
            if let item = store.currentItem {
                editor.bind(to: item)
            }
        } else {
            // User was editing: keep buffer. But if the file disappeared, reflect it.
            if let id = editingItemID,
               let key = store.selectedSourceKey,
               store.itemsBySource[key]?.contains(where: { $0.id == id }) == false
            {
                // External delete during edit — leave buffer in memory; status bar
                // will show "未保存" because boundItem still points to gone file.
                // On ⌘S, save will throw and surface error alert.
            }
        }
    }
}
```
